# 📁 Сортировщик файлов для Windows

Учебный проект по разработке интеллектуального программного агента для автоматизированного управления файловой структурой. Приложение предоставляет полноценный графический интерфейс для визуального контроля и аналитики данных.

## 🛠 Технологии

| Компонент | Технология |
| :--- | :--- |
| **GUI Framework** | Streamlit |
| **Data Processing** | Pandas |
| **Visualization** | Plotly |
| **System API** | Pathlib, Shutil |
| **Reporting** | HTML5 / CSS3 |

## 🚀 Быстрый старт


1. **Клонирование репозитория**
```
git clone https://github.com/Matyuzha/python_project.git
cd file_sorter
```

2. **Установка необходимых библиотек**
```
pip install -r requirements.txt
```

3. **Запуск программы**
```
streamlit run sort_files_ready.py
```

📋 Системные требования

Python — версия 3.8 или выше 

ОС — Windows (для корректной работы tkinter диалогов) 

Библиотеки — Streamlit, Pandas, Plotly, Psutil
