# 📁 Сортировщик файлов для Windows

Учебный проект по разработке интеллектуального программного агента для автоматизированного управления файловой структурой. [cite_start]Приложение предоставляет полноценный графический интерфейс для визуального контроля и аналитики данных[cite: 242, 243].

## 🛠 Технологии

| Компонент | Технология |
| :--- | :--- |
| **GUI Framework** | [cite_start]Streamlit [cite: 218] |
| **Data Processing** | [cite_start]Pandas [cite: 213] |
| **Visualization** | [cite_start]Plotly [cite: 215] |
| **System API** | [cite_start]Pathlib, Shutil [cite: 214, 217] |
| **Reporting** | [cite_start]HTML5 / CSS3 [cite: 226] |

## 🚀 Быстрый старт

1. **Клонирование репозитория**
```bash
git clone [https://github.com/ваш_логин/file_sorter.git](https://github.com/ваш_логин/file_sorter.git)
cd file_sorter


2. **Установка необходимых библиотек**
pip install -r requirements.txt


3. **Запуск программы**
streamlit run sort_files_ready.py


📋 Системные требования

Python — версия 3.8 или выше 


ОС — Windows (для корректной работы tkinter диалогов) 

Библиотеки — Streamlit, Pandas, Plotly, Psutil
